## Problem Statement
This project focuses on converting black and white images into colored images using deep learning techniques.

Kaggle Image Colorization Dataset (https://www.kaggle.com/datasets/shravankumar9892/image-colorization) 

## Model / Technique Used
CNN / U-Net based image colorization approach.

## How to Run
1. Open the notebook file (.ipynb) in Google Colab or Jupyter Notebook.
2. Run all cells to see training and results.

## Results
The model produces visually improved colored images from grayscale inputs.
